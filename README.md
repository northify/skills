# GapQuery Skills

Agent skills for [GapQuery](https://gapquery.com) — discover gaps and opportunities across 11 app ecosystems with 35,600+ apps.

## Available Skills

| Skill | Description |
|-------|-------------|
| `gapquery-research` | Guided discovery and deep research workflow for app marketplace opportunities |

## Install

```bash
npx skills add gapquery/skills --skill gapquery-research
```

## Prerequisites

You need a GapQuery account with an API key and the MCP server configured. See the [setup guide](https://gapquery.com/docs/mcp-setup) for details.
